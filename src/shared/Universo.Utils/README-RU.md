# Universo.Utils

Вспомогательные функции и помощники для Universo Platformo C#.

## Обзор

Эта библиотека предоставляет общие вспомогательные функции, методы расширения и вспомогательные классы, используемые в экосистеме Universo Platformo.

## Содержимое

- **Методы расширения**: Методы расширения C# для общих операций
- **Вспомогательные классы**: Утилитные классы для общих задач
- **Валидация**: Проверка и санитизация входных данных
- **Конвертеры**: Конвертеры типов и форматеры
- **Защита**: Защитные проверки аргументов

## Использование

```csharp
using Universo.Utils;

// Примеры использования будут добавлены по мере реализации утилит
```

## Зависимости

- .NET 9.0
- Universo.Types (внутренняя ссылка)
