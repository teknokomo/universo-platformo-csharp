# Universo.Utils

Utility functions and helpers for Universo Platformo C#.

## Overview

This library provides common utility functions, extension methods, and helper classes used across the Universo Platformo ecosystem.

## Contents

- **Extension Methods**: C# extension methods for common operations
- **Helper Classes**: Utility classes for common tasks
- **Validation**: Input validation and sanitization
- **Converters**: Type converters and formatters
- **Guards**: Argument guard clauses

## Usage

```csharp
using Universo.Utils;

// Example usage will be added as utilities are implemented
```

- .NET 9.0
- Universo.Types (internal reference)
