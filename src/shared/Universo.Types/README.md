# Universo.Types

Shared type definitions and models for Universo Platformo C#.

## Overview

This library contains common types, interfaces, and data models used across all packages in the Universo Platformo ecosystem.

## Contents

- **Common Types**: Shared value objects and primitives
- **Entity Models**: Base entity classes and interfaces
- **DTOs**: Data Transfer Objects for API communication
- **Enums**: Enumeration types used across the platform
- **Constants**: Shared constant values

## Usage

```csharp
using Universo.Types;

// Example usage will be added as types are implemented
```

## Dependencies

- .NET 9.0
- No external dependencies (by design)
