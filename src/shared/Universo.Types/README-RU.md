# Universo.Types

Общие определения типов и модели для Universo Platformo C#.

## Обзор

Эта библиотека содержит общие типы, интерфейсы и модели данных, используемые во всех пакетах экосистемы Universo Platformo.

## Содержимое

- **Общие типы**: Общие объекты значений и примитивы
- **Модели сущностей**: Базовые классы сущностей и интерфейсы
- **DTO**: Объекты передачи данных для API коммуникации
- **Перечисления**: Типы перечислений, используемые на платформе
- **Константы**: Общие константные значения

## Использование

```csharp
using Universo.Types;

// Примеры использования будут добавлены по мере реализации типов
```

## Зависимости

- .NET 9.0
- Без внешних зависимостей (по дизайну)
