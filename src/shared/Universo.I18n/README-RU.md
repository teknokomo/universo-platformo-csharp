# Universo.I18n

Поддержка интернационализации для Universo Platformo C#.

## Обзор

Эта библиотека обеспечивает возможности интернационализации (i18n) и локализации (l10n) для экосистемы Universo Platformo, с основной поддержкой английского и русского языков.

## Содержимое

- **Файлы ресурсов**: Локализованные строковые ресурсы
- **Провайдеры культуры**: Определение культуры и языка
- **Форматеры**: Форматирование даты, времени и чисел
- **Сервисы перевода**: Поддержка перевода во время выполнения
- **Языковые расширения**: Вспомогательные методы для локализации

## Поддерживаемые языки

- Английский (en)
- Русский (ru)

## Использование

```csharp
using Universo.I18n;

// Примеры использования будут добавлены по мере реализации функций i18n
```

## Зависимости

- .NET 9.0
- Microsoft.Extensions.Localization
