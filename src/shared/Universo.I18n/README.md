# Universo.I18n

Internationalization support for Universo Platformo C#.

## Overview

This library provides internationalization (i18n) and localization (l10n) capabilities for the Universo Platformo ecosystem, with primary support for English and Russian languages.

## Contents

- **Resource Files**: Localized string resources
- **Culture Providers**: Culture and language detection
- **Formatters**: Date, time, and number formatting
- **Translation Services**: Runtime translation support
- **Language Extensions**: Helper methods for localization

## Supported Languages

- English (en)
- Russian (ru)

## Usage

```csharp
using Universo.I18n;

// Example usage will be added as i18n features are implemented
```
